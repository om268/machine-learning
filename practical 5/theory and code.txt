1. Simple Linear Regression

ğŸ“Œ Problem: Predict a student's score based on the number of study hours.

ğŸ§ª Code:

import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression
import numpy as np

# Sample data: Study Hours vs Score
X = np.array([1, 2, 3, 4, 5]).reshape(-1, 1)  # Independent variable (Hours)
y = np.array([10, 20, 30, 40, 50])           # Dependent variable (Score)

# Create and train the model
model = LinearRegression()
model.fit(X, y)

# Predict
y_pred = model.predict(X)

# Output results
print("Coefficient (slope):", model.coef_[0])
print("Intercept:", model.intercept_)

# Plot
plt.scatter(X, y, color='blue', label='Actual')
plt.plot(X, y_pred, color='red', label='Predicted Line')
plt.xlabel('Study Hours')
plt.ylabel('Score')
plt.title('Simple Linear Regression')
plt.legend()
plt.show()

2. Multiple Linear Regression

ğŸ“Œ Problem: Predict house price based on area, number of bedrooms, and age of the house.

ğŸ§ª Code:

import pandas as pd
from sklearn.linear_model import LinearRegression

# Sample data
data = {
    'Area': [1000, 1500, 2000, 2500, 3000],
    'Bedrooms': [2, 3, 4, 4, 5],
    'Age': [5, 10, 15, 20, 25],
    'Price': [300000, 400000, 500000, 600000, 650000]
}

df = pd.DataFrame(data)

# Features and target
X = df[['Area', 'Bedrooms', 'Age']]  # Independent variables
y = df['Price']                      # Dependent variable

# Create and train the model
model = LinearRegression()
model.fit(X, y)

# Predict
predictions = model.predict(X)

# Output results
print("Coefficients:", model.coef_)
print("Intercept:", model.intercept_)

# Display predicted prices
df['Predicted Price'] = predictions
print(df)
